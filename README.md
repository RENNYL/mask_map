# 簡介
因疫情導致口罩需求增加，因此藉機練習AJAX串接衛服部健保署釋出API。


# 使用技術
<ul>
      <li>使用 leaflet.js 與 OpenStreetMap 搭建地圖資料</li>
      <li>透過 JavaScript AJAX串接API</li>
      <li>使用 Leaflet.markercluster 群集化資料</li>
      <li>縣市／鄉鎮區／地址搜尋功能</li>
</ul>
